Team Russell – IFT Coursework One
===================================

An **8-metric Value + Quality composite factor data pipeline** for the UCL IFT
Big Data in Quantitative Finance coursework.

The pipeline ingests market and financial data for a 678-company investable
universe, computes sector-neutral composite factor scores using an
inverse-normal z-score methodology, and stores them in PostgreSQL for
portfolio construction in Coursework Two.
